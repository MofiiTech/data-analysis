# Chapter 3 - Built-in Data Structures, Functions and Files

## 3.1 Data Structures and Sequences

### Tuple

* Basics:

  - fixed-length
  - immutable

* Create a basic tuple:

  ```Python
  In [1]: tup = 4, 5, 6
  
  In [2]: tup
  Out[2]: (4, 5, 6)
  ```

* Create a nested tuple:

  ```Python
  nested_tup = (4, 5, 6), (7, 8)
  ```
